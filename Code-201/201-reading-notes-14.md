## CSS Transitions & Animations

What does a CSS transition allow the developer to do to an element?

To alter the appearance and behavior of an element when it is active, hovered, ETC.

How does a CSS animation differ from a CSS transition?

A transition is best for single state changes and an animation is best for when there are multiple states.
