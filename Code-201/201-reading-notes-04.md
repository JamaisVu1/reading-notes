## CSS

1. What is meant by “normal flow”?

How elements are displayed without having done any CSS.

2. What are a few differences between block-level and inline elements?

Block-level fills the space of the parent element, while inline size is based off the content. Also, block-levels have a different flow direction. Descending down the page, instead of sitting on one line like a inline.

3. ___ positioning is the default for every html element.

Static positioning.

4. Name a few advantages to using absolute positioning on an element.

To change an element outside of the other elements flow.

5. What is a key difference between fixed positioning and absolute positioning?

Fixed positioning places the element on one part of the screen, like a navigation bar.
